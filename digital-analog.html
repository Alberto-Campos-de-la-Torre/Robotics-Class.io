Más Allá del Encendido/Apagado

El mundo real raramente es solo blanco o negro. La luz del sol cambia gradualmente, el volumen de la música tiene muchos niveles, la temperatura sube y baja de forma continua. Hoy aprenderemos cómo nuestros microcontroladores pueden "sentir" estos matices analógicos.

Recordatorio: Digital vs. Analógico

Señal Digital

Solo tiene dos estados posibles: ALTO (HIGH) o BAJO (LOW). Es discreta, como un interruptor.

Lectura: Usamos digitalRead().

Señal Analógica

Puede tomar cualquier valor dentro de un rango continuo. Es fluida, como una onda de sonido.

Lectura: Usaremos analogRead().

"Sintiendo" el Mundo: Entradas Analógicas

Leyendo Voltajes Continuos

Mientras que una entrada digital solo ve 0V (LOW) o 5V (HIGH), una entrada analógica puede leer cualquier voltaje intermedio. Esto es crucial para sensores que miden magnitudes físicas que varían continuamente, como la luz, la temperatura o la posición.

El microcontrolador usa un Convertidor Analógico-Digital (ADC) para "traducir" este voltaje continuo a un número que pueda entender.

Resolución del ADC: El Arduino UNO tiene un ADC de 10 bits. Esto significa que puede representar el voltaje de entrada (0V a 5V) con 210 = 1024 niveles diferentes. El valor leído estará entre 0 (para 0V) y 1023 (para 5V).
La Función `analogRead()`

Usamos esta función para obtener ese número (0-1023) que representa el voltaje en un pin analógico.

const int sensorPin = A0; // Define el pin analógico a usar (A0) void setup() { Serial.begin(9600); // Inicia comunicación serial para ver los valores pinMode(sensorPin, INPUT); // Aunque es analógico, se configura como INPUT } void loop() { int valorSensor = analogRead(sensorPin); // Lee el pin A0 Serial.print("Valor del sensor: "); Serial.println(valorSensor); // Muestra el valor (0-1023) en el Monitor Serie delay(200); // Pequeña pausa para no saturar el monitor }

¿Cómo Funciona un ADC por Dentro?

El Convertidor Analógico-Digital (ADC) es un chip fascinante. Su trabajo es tomar una señal que varía suavemente (como el voltaje de un sensor) y convertirla en una serie de números discretos que la computadora puede procesar. Lo hace en tres pasos principales:

1. Muestreo (Sampling)

El ADC toma "fotografías" del voltaje analógico a intervalos de tiempo muy regulares y rápidos. Cada fotografía captura el nivel de voltaje en ese instante preciso.

2. Cuantificación (Quantization)

Como el ADC solo puede representar un número finito de niveles (1024 en Arduino), cada muestra tomada se "redondea" al nivel discreto más cercano disponible. Aquí es donde se pierde un poco de información (ruido de cuantificación), pero es inevitable.

3. Codificación (Encoding)

Finalmente, ese nivel discreto asignado se convierte en un número binario. Con 10 bits, podemos representar números desde 0000000000 (0 decimal) hasta 1111111111 (1023 decimal). Este número binario es el que la función analogRead() nos entrega.

En resumen, el ADC "digitaliza" la realidad analógica tomando muestras rápidas, asignándolas a niveles predefinidos y convirtiendo esos niveles a números binarios.

Actividades Prácticas con Sensores Analógicos

Actividad 1: Leyendo un Potenciómetro

Un potenciómetro es una resistencia variable que podemos girar. Al conectarlo correctamente, actúa como un divisor de voltaje, dándonos una señal analógica proporcional al ángulo de giro.

Conexión:

  • Pin izquierdo a GND.
  • Pin central (cursor) a A0.
  • Pin derecho a 5V.

Tarea: Escribe un programa que lea el valor del potenciómetro (A0) y lo muestre en el Monitor Serie. Observa cómo cambia el valor (0-1023) al girar la perilla.

Pasos:
  1. Construye el circuito en Tinkercad.
  2. Escribe el código usando analogRead(A0) y Serial.println().
  3. Abre el Monitor Serie en Tinkercad y simula. Gira el potenciómetro.
  4. (Opcional) Construye el circuito físico y pruébalo.

Actividad 2: Sensor de Luz (Fotoresistencia LDR)

Una fotoresistencia (LDR) cambia su resistencia según la cantidad de luz que recibe (menos resistencia = más luz). Para leerla, la usamos en un divisor de voltaje con una resistencia fija (ej. 10kΩ).

Conexión (Divisor de Voltaje):

  • 5V a una pata de la LDR.
  • La otra pata de la LDR a A1 Y a una pata de la resistencia de 10kΩ.
  • La otra pata de la resistencia de 10kΩ a GND.

Tarea: Escribe un programa que lea el valor del pin A1 y lo muestre en el Monitor Serie. En Tinkercad, puedes simular diferentes niveles de luz haciendo clic en la LDR.

Pasos:
  1. Construye el circuito en Tinkercad.
  2. Escribe el código similar al anterior, pero leyendo A1.
  3. Simula y ajusta la luz en el simulador para ver cómo cambian los valores (¿valores más altos significan más o menos luz?).
  4. (Opcional) Construye el circuito físico y prueba tapando la LDR.

Reto: Luz Nocturna Simple

Usando el circuito de la fotoresistencia (Actividad 2) y un LED conectado al pin 13, modifica el código para que el LED se encienda (digitalWrite(13, HIGH)) solo cuando el valor leído del sensor de luz (A1) sea bajo (ej. menor a 300), y se apague (digitalWrite(13, LOW)) cuando sea alto.

Pasos:
  1. Combina los circuitos de la LDR y el LED en Tinkercad.
  2. Escribe el código: Dentro del loop(), lee A1 con analogRead(), usa una estructura if/else para comparar el valor leído con un umbral (ej. 300), y enciende o apaga el LED en el pin 13 según corresponda.
  3. Simula y verifica que el LED se encienda al bajar la luz simulada sobre la LDR.
  4. (Opcional) Construye el circuito físico.
