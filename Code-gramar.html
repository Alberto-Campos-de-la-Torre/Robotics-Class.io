La Gramática del Código

Así como un idioma tiene reglas gramaticales para formar oraciones, un lenguaje de programación tiene una sintaxis que debemos seguir. Hoy aprenderemos las reglas esenciales del C++, el lenguaje que da vida a nuestros robots y circuitos.

Almacenando Información: Variables y Tipos de Datos

Una variable es un contenedor con nombre para guardar datos. Cada pieza de información en un programa tiene un tipo, que le dice al microcontrolador cuánta memoria usar y qué clase de datos guardar. Los más importantes son:

  • int (Entero): Para números enteros sin decimales, positivos o negativos. Ocupa 2 bytes en un Arduino UNO (rango de -32,768 a 32,767).
    int numeroDePasos = 100;
  • float (Flotante): Para números con decimales. Son más precisos pero también usan más memoria (4 bytes).
    float voltaje = 4.85;
  • bool (Booleano): Solo puede tener dos valores: true (verdadero) o false (falso). Es la base de toda la lógica y las decisiones.
    bool sensorActivado = false;
  • char (Carácter): Almacena un único carácter, como una letra o un símbolo. Se escribe entre comillas simples.
    char inicial = 'A';
  • String (Cadena de texto): Para guardar palabras o frases completas. Es un tipo de dato más complejo pero increíblemente útil. Se escribe entre comillas dobles.
    String mensaje = "Sistema Activado";

Reglas Esenciales del Lenguaje (Sintaxis)

Punto y Coma (;)

Cada instrucción o "frase" en C++ debe terminar con un punto y coma. Es como el punto final de una oración. Olvidarlo es uno de los errores más comunes.

Llaves ({ })

Las llaves se usan para agrupar instrucciones en bloques de código. Todo lo que está dentro de las llaves de una función o un condicional pertenece a ese bloque.

Comentarios (//)

Son notas para los humanos que el compilador ignora. Se usan para explicar qué hace el código, lo que es vital para entender programas complejos.

Tabulación

El código dentro de un bloque de llaves {} se indenta (se le da un espacio a la izquierda). Esto no afecta al programa, pero es crucial para que los humanos puedan leerlo fácilmente.

Controlando el Flujo del Programa

Condicionales: if / else

Permiten que tu programa tome diferentes caminos según una condición. Es la herramienta principal para la toma de decisiones.

Plan (Pseudocódigo)
SI temperatura > 25 ENTONCES   ESCRIBIR "Hace calor" SINO   ESCRIBIR "No hace calor" FIN SI
Código (C++)
if (temperatura > 25) {   Serial.println("Hace calor"); } else {   Serial.println("No hace calor"); }

Bucles: for y while

Se usan para repetir un bloque de código. El bucle for es ideal para repetir un número conocido de veces, y while para repetir mientras se cumpla una condición.

Plan (Pseudocódigo)
// Bucle PARA PARA i=0 HASTA 2 HACER   ESCRIBIR "Hola" FIN PARA // Bucle MIENTRAS MIENTRAS boton_no_presionado HACER   ... esperar ... FIN MIENTRAS
Código (C++)
// Bucle for for (int i = 0; i < 3; i++) {   Serial.println("Hola"); } // Bucle while while(digitalRead(pinBoton) == LOW) {   // ... esperar ... }

Actividad Práctica: De Pseudocódigo a Código

Problema: Alarma de Temperatura

Vamos a diseñar el código para una alarma simple. Un sensor de temperatura nos da un valor. Si la temperatura supera los 40 grados, un LED rojo de alarma debe encenderse. Si está por debajo de 10 grados, un LED azul de advertencia debe encenderse. Si está entre esos valores, ambos LEDs deben estar apagados.

1. El Plan (Pseudocódigo)

INICIO   MIENTRAS verdadero HACER     Definir temperatura Como Entero     temperatura = LEER SENSOR        SI temperatura > 40 ENTONCES       ENCENDER LED ROJO       APAGAR LED AZUL     SINO SI temperatura < 10 ENTONCES       APAGAR LED ROJO       ENCENDER LED AZUL     SINO       APAGAR LED ROJO       APAGAR LED AZUL     FIN SI   FIN MIENTRAS FIN

2. Tu Tarea: Escribe el Código C++

Traduce el plan a código de Arduino. Usa int temperatura = analogRead(A0) / 4; para simular una lectura de 0 a 100 grados. Asigna el LED rojo al pin 13 y el LED azul al pin 12.

3. Solución Sugerida (Código C++)

const int ledRojoPin = 13; const int ledAzulPin = 12; const int sensorPin = A0; void setup() { pinMode(ledRojoPin, OUTPUT); pinMode(ledAzulPin, OUTPUT); pinMode(sensorPin, INPUT); } void loop() { int lecturaSensor = analogRead(sensorPin); int temperatura = lecturaSensor / 4; // Simula 0-100 grados if (temperatura > 40) { digitalWrite(ledRojoPin, HIGH); digitalWrite(ledAzulPin, LOW); } else if (temperatura < 10) { digitalWrite(ledRojoPin, LOW); digitalWrite(ledAzulPin, HIGH); } else { digitalWrite(ledRojoPin, LOW); digitalWrite(ledAzulPin, LOW); } }
