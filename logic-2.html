Simulación del Mecanismo de Seguridad

Ahora, vamos a aplicar lo aprendido simulando el circuito del Ejercicio 1 (Mecanismo de Seguridad) en Tinkercad. Utiliza los circuitos integrados de compuertas lógicas (como el 7486 para XOR y 7408 para AND) para construir la ecuación que formulaste.

Objetivos:
  • Construir el circuito lógico para la ecuación: Salida = (A XOR B) AND C.
  • Usar interruptores para simular las entradas (A, B, C).
  • Usar un LED como indicador de la salida (el mecanismo de seguridad).
  • Verificar que el comportamiento del circuito simulado coincida con tu tabla de verdad.
